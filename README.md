# Lectio

A speed reading app using RSVP (Rapid Serial Visual Presentation) with ORP (Optimal Recognition Point) alignment.

## Features

- Paste any text and read one word at a time
- Adjustable WPM (100-1000)
- ORP letter highlighted and centered for optimal recognition
- Context view to see surrounding words
- Progress saved to localStorage
- Keyboard controls (Space, Arrows, C, Esc)

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- React Router
- localStorage for persistence
